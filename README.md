# LLM Behavior & Prompt Engineering Study

**Research Study by Sneha Kumari J**
B.Tech Artificial Intelligence & Machine Learning | Presidency University, Bengaluru

---

## Overview
This research study investigates how different prompt strategies, 
temperature settings, context lengths, and hallucination patterns 
affect the output quality of Large Language Models (LLMs).

---

## Experiments Conducted

### Experiment 1 — Prompt Structure Analysis
- Vague vs Structured vs Few-Shot prompting
- Key Finding: Structured prompts produce significantly more focused responses

### Experiment 2 — Temperature Analysis
- Low (0.1) vs Medium (0.7) vs High (1.5) temperature
- Key Finding: Lower temperature increases consistency, higher temperature increases creativity

### Experiment 3 — Context Length Analysis
- No context vs Medium context vs Detailed context
- Key Finding: Detailed context dramatically improves response accuracy

### Experiment 4 — Hallucination Detection
- Tested LLM responses on fake people, fake events, ambiguous facts
- Key Finding: LLMs confidently hallucinate about non-existent entities

## Tech Stack

---

## Key Findings
1. Prompt structure is the single biggest factor affecting output quality
2. Temperature directly controls creativity vs consistency tradeoff
3. Context length significantly improves factual accuracy
4. LLMs are prone to hallucination on unverifiable claims
